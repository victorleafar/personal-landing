# Personal Landing

Landing page simples e responsiva para personal trainers, criada para apresentar serviços e captar contatos de forma objetiva.

## Demo

Projeto em produção:  
https://personal-landing-lilac.vercel.app/

## Sobre o projeto

Este projeto consiste em uma landing page voltada para personal trainers que desejam divulgar seus serviços de forma profissional.  
A página contém seções básicas como apresentação, sobre, programas de treino, depoimentos e formulário de contato.

O foco do projeto é layout, responsividade e organização visual.

## Funcionalidades

- Seção inicial com chamada para ação  
- Apresentação do personal trainer  
- Lista de programas de treino  
- Depoimentos de alunos  
- Formulário de contato  
- Layout responsivo para desktop e mobile  

## Tecnologias

- React  
- Vite  
- CSS
